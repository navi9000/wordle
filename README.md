# О проекте

Финальный проект из книги "React in Depth", в котором реализована широко известная игра с отгадыванием слов из 5 букв.

## Запуск проекта

- npm run dev: запуск проекта для разработки
- npm run build: сборка проекта
- npm run storybook: запуск storybook

## Особенности проекта

- Проект создан с помощью vite
- Логика игры реализована при помощи машины состояний xstate
- Элементы интерфейса созданы отдельно с помощью storybook
